📌 Project Overview

Modex Bags is a clean, minimal luxury product landing page showcasing a premium men's leather bag collection.

The project demonstrates:

Responsive layout using Tailwind CSS

Smooth scroll interaction using vanilla JavaScript

Dynamic quantity selection

E-commerce functionality powered by Snipcart

Local development configuration

Deployment-ready setup for Netlify

Although built alongside a guided course, I configured, debugged, and deployed the full integration independently.

🔧 Local Development Setup

Clone the repository

Install dependencies:

npm install

Build Tailwind CSS:

npx tailwindcss -i ./styles/input.css -o ./dist/output.css --watch

Run with Live Server

Ensure Snipcart test API key is configured
